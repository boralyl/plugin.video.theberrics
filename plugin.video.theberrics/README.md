xbmc :: plugin.video.theberrics
===============================

XBMC video addon that allows browsing and watching videos from theberrics.com

Supported Channels

* Bangin!
* Battle Commander
* DIY or DIE
* First Try Fridays
* General Ops (slow loading, need to implement caching)
* Off the Grid
* Process
* Recruit
* Shoot All Skaters
* Thrashin' Thursdays
* Trajectory
* VHS
